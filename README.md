
## Description

A simple REST API application under Docker environment.
* NestJS
* TypeORM
* PostgreSQL
* Swagger
* Docker


## Run with Docker

Install docker and docker-compose using the instruction https://docs.docker.com/get-docker/

Run command:

```
docker-compose up
```

## Run with NPM

Installing NPM modules

```
npm install
```
Running application

```
npm run start:dev
```
